Helix Server - Un framework de servidor web ligero, de alto rendimiento, totalmente extensible y minimalista para Bun.js.

## Características

- **Ligero y Rápido**: Construido sobre Bun.js para un rendimiento óptimo.
- **Extensible**: Agrega fácilmente middleware y plugins para mejorar la funcionalidad.
- **Minimalista**: API simple y limpia para un desarrollo rápido.
- **Soporte TypeScript**: Totalmente tipado para una mejor experiencia de desarrollo.
- **Enrutamiento**: Sistema de enrutamiento flexible para manejar varios métodos y rutas HTTP.
- **Soporte de Middleware**: Integra middleware personalizado para el procesamiento de solicitudes.
- **Código Abierto**: Impulsado por la comunidad con contribuciones bienvenidas.
